## 项目目标

- 展示纯前端技术的实践案例
- 提供简单、易懂的代码示例
- 研究试验效果

## 如何使用

1. 克隆仓库
2. 全局安装一个http-server `npm install -g http-server`
3. 在项目目录下 `http-server` 命令启动服务器
4. 点击相应链接查看不同的效果
