## 技术决策
| 领域          | 技术选择                  | 理由                          |
|---------------|-------------------------|-------------------------------|
| 核心框架       | 原生 JavaScript         | 零依赖、高执行效率               |
| 状态管理       | 观察者模式               | 低耦合、易扩展                  |
| 样式方案       | CSS 变量 + Flex 布局     | 主题切换友好、响应式支持          |
| 构建方案       | ES Modules              | 现代浏览器原生支持                |

## 关键设计
1. **双向数据流**：UI事件 → 核心逻辑 → 显示更新
2. **错误边界**：独立错误处理管道
3. **主题系统**：CSS变量动态注入机制
